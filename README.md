# Likey Wireframe

`와이어프레임 생성` 폴더의 Vite/React 와이어프레임 번들을 기준으로 루트 프로젝트 초기 구조를 구성한 저장소입니다.

## 시작하기

```bash
npm install
npm run dev
```

## 현재 구조

```text
.
├── index.html
├── package.json
├── src
│   ├── app
│   ├── imports
│   ├── main.tsx
│   └── styles
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```

## 참고

- 원본 와이어프레임 번들은 `와이어프레임 생성/` 폴더에 그대로 보관했습니다.
- 실제 앱 작업은 루트 기준 구조에서 이어가면 됩니다.
